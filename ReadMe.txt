# Fitness-Club-Management-Club



The Fitness-Club-Management-Club program allows the user to manage the membership accounts for a fitness club.

Current functionality includes:
	
Asking how many new members will be added to the database.

Prompting user to enter name and age for each member

Member given 5 digit membership number with first digits being their age

Business points:
A member can enrole in the following classes:
- body pump
- yoga
- LES MILIS GRIT
- Spin

Each class cost £30 to enrole

Member should be able to see their balance

Member should be able to see their status, Name, ID, courses enrolled and balance


## Getting Started



These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.



### Prerequisites



Ensure you have Java 8 or higher installed on your machine. 
To check, run the following on the command line:


`java --version`8



If you do not have Java installed, you should get a prompt to do so. 
Once installed,
 
you should also have access to `javac`. You can check this by running:

`javac --version`8



### Running Fitness-Club-Management-System



Run these commands in your terminal to test the program is working correctly.




```


javac FitnessClubManagementApp.java

java Memeber.java

```
